## Inspiration
My inspiration to create this project was the great amount of effort that I personally put when it comes to study for Midterms and Finals, I realise that my attention span is very short unless there is something "entertaining" that I could watch, so my goal here was to try to mix both worlds, to see if I could find a balance between distraction and focus.
## What it does
The App allows the user to answer various quizzes while also watching videos that are very popular in social media, the goal here is to keep the user focus on the questions and not making him leave the screen to do other thing.
## How we built it
I used Figma to create the design, React, JavaScript ,CSS, HTML and Gemini API.
## Challenges we ran into
I honestly spend too much time trying to get the desired design, working with CSS made me realise that it is really challenging to get app to be responsive and also I ran into the problem of separating sections of the design, which lead me to modify other sections by accident and also the implementation of the API, the API was hard to test specially with the limited amount of tries I can do wwith the FREE trial.  
## Accomplishments that we're proud of
I am proud of the implementation of the quizzes of various subjects from  Mathematics, Computers or History, as well as the custom subject that the user can select or type depending of what he wants to learn. 
## What we learned
I learned a lot about Figma and its plugings, I learned more deeply about the design with CSS, I learned about React and it's multiple uses.
## What's next for CANLearn
In the future I would like to continue working on this project because I feel that the idea could help many people that struggle to focus when it comes to academics or any other subjects that they may try to learn.
